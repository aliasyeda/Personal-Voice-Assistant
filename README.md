# Personal-Voice-Assistant



Python voice assistant with text-to-speech. Type commands, hear voice responses.

## Features

- 🎤 Text-to-speech output
- ⏰ Time, date, day
- 📂 Open apps (Notepad, Calculator, Chrome, Paint)
- 🌐 Google search & YouTube
- 😂 Random jokes
- ⌨️ Keyboard input

## Installation


pip install pywin32

## Usage

python voice_assistant.py
## Commands
Command	  Action
time	Current time
date	Current date
notepad	Open Notepad
calculator	Open Calculator
chrome	Open Chrome
search	Google search
youtube	Open YouTube
joke	Tell a joke
help	Show commands
exit	Close assistant
## Example

You: time

Assistant: The time is 02:30 PM

You: joke  

Assistant: Why do programmers prefer dark mode? Because light attracts bugs!
## Dependencies
pywin32 - Windows Text-to-Speech
